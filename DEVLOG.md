# 🎨 SoundCanvas — 개발 로그

## 프로젝트 개요
마이크 입력을 실시간 분석하여 소리 유형(목소리, 박수, 핑거스냅, 웃음)에 따른 제너러티브 아트를 캔버스에 그리는 웹 애플리케이션.

---

## ✅ 완료된 작업

### 핵심 엔진
- **오디오 분석**: Web Audio API + YAMNet(MediaPipe) 기반 실시간 소리 분류
- **제너러티브 엔진**: Perlin noise 기반 유기적 시각화, 소리 유형별 고유 그래픽
- **무음 처리**: 말을 하지 않을 때 모든 이동/생성이 완전히 중단됨

### 목소리 시각화
- 흐름 라인(flow line) — smooth, dotted 스타일의 유기적 궤적
- 점묘(stipple), 나선(spiral), 성운(nebula) 등 부가 그래픽 요소
- **피치 → 방향 매핑**: 고음이면 좌로, 저음이면 우로 라인 진행 방향 제어
- 피치 방향 민감도 튜닝 파라미터 추가

### 스냅(핑거스냅) 시각화
- 크리스탈 스타버스트 + 확장 링 + 아이스 샤드
- **크랙 라인 추가**: 지그재그 결정 균열 효과
- 미세 점묘(stipple)

### 박수 시각화
- 충격파 링 + 중앙 글로우
- **방사형 스트릭 라인 추가**: 충격파 줄기 효과
- 불규칙 블롭 형태의 페인트 스플래터

### 웃음 시각화
- 그라디언트 버블 + 하이라이트 아크
- **컨페티 추가**: 떨어지는 작은 색종이 조각
- **바운싱 아크 트레일 추가**: 곡선 궤적
- 점 산포 + 미니 나선

### 튜닝 시스템
- 분류 감도, 각 소리 유형별 시각화 파라미터를 실시간 슬라이더로 조절
- 파라미터 JSON 복사 기능
- **localStorage 영구 저장**: 새로고침해도 설정값 유지

### 기본 설정값
- 마이크 감도: 0.4 / 반응 임계값: 0.06
- 라인 굵기 배율: 2.4 / 커서 속도: 1.1
- 점묘 확률: 0.21 / 점묘 크기: 37
- 피치 방향 민감도: 1.8
